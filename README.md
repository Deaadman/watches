## Features
Watches is a modification which alters how time is told within [**The Long Dark**](https://www.hinterlandgames.com/the-long-dark/).

<details>
    <summary>General Features</summary>

- **Digital Watch:** A watch which tells the precise time, but with the downfall of a battery.
- **Analogue Watch:** A watch that tells the precise time, in analogue format.
- **Stopwatch:** Similar to the analogue watch, just not updated constantly.
- **Sundial:** An item alternative to watches with limited capabilities.
</details>

<details>
    <summary>Gameplay Mechanics</summary>

- **Aurora:** The aurora can affect the display of the digital time, making it unusable - however it charges it in the process.
- **Condition:** If an items condition is too low, it begins to partially display the time - requiring to user to maintain their time pieces.
- **Freezing:** If any watch item gets over a certain freezing threshold, then the time freezes in place.
- **Battery:** The Digital Watch has a battery, which is drained throughout its use.
</details>

<details>
    <summary>UI Features</summary>

- **Digital Time Indicator:** When wearing a digital watch, it now displays the current time down to the minute.
- **Analogue Time Indicator:** When wearing an analogue watch, or if a stopwatch is used - it displays a fully functional analogue clock.
- **Vanilla TimeWidget Overhaul:** This is now disabled by default, and is only shown whenever a sundial is used.
- **Responds to Mechanics** There are different mechanics that can influence how the time is displayed throughout the game.
</details>

## Compatibility

### Required Dependencies
For this mod to work, ensure you have the following mods below:

- [**ModComponent**](https://github.com/dommrogers/ModComponent)
- [**ModData**](https://github.com/dommrogers/ModData)
- [**Modders' Gear Toolbox**](https://github.com/Jods-Its/Modders-Gear-Toolbox)

## Installation

1. [**Download**](https://github.com/LavaGang/MelonLoader/releases/latest/download/MelonLoader.Installer.exe) MelonLoader and install it into your game.
2. [**Download**](https://github.com/Deaadman/watches/releases/latest/download/Watches.zip) the latest version of this mod.
3. Navigate to where your game's directory.
4. Extract the `Watches.zip` you downloaded and paste the contents into the `mods` folder of your game's directory.
5. Launch the game.
